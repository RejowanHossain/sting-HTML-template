C) How To Edit Figma File - top

To edit the template, you need to open the downloaded file you need to go to 'Befeis_Figma_Main_File' . Go inside the Assets folder. As you can see on the bottom, there's a images folder. Go inside the folder and replace your image with placeholders. Remember to rename your images as the placeholders name.
